Neko Check setup guide

Welcome to the Neko Check Setup guide!
Neko Check is entirely developer by Luiskawaiineko and it doesn't use any external code
Remember that Neko Check is COMPLETELY FREE!
Here you will find information on how to setup NekoCheck.
Anyways it's made to be easy to setup.

Adding Neko Check to your game


To add Neko Check in your game you need to download the model.
You can do this in two ways:

  • Model download via Roblox Studio

    1. Open the toolbox inside Roblox Studio
    2. Search "Neko Check"
    3. Click the original model made by Luiskawaiineko
    4. Click the model in the explorer
    5. Press CTRL + U or Left click on the model and 'Ungroup'
    6. Done! Now you need to setup it

  • Model download via this website

    1. Click here to download the model
    2. Open the Explorer inside Roblox Studio
    3. Click in the workspace
    4. Right Click and 'Insert from file'
    5. Click the downloaded model
    6. Press CTRL + U or Left click on the model and 'Ungroup'
    7. Done! Now you need to setup it

Setting up doors and screens


The kit comes with a screen and two doors.
These screen and doors are models.
WARNING! DO NOT GROUP OR UNGROUP THE DOORS NOR THE SCREENS!
DOING SO WILL MAKE IT DON'T WORK.
To add doors or screens, just Duplicate the existing ones
Delete all doors that aren't needed. To duplicate them, right click them in the Explorer and click 'Duplicate' or do CTRL + D
The doors need to be set up.
To do that, Expand the model in the explorer and set it number and it's Type (Ex: Standard, Suite...).

Configuration


There is a 'configuration' folder inside the NekoCheck Kit folder
In the below list you can find what does each one do.
  • BackgroundImage:
    Add a decal ID and it will be placed as background in the screen when it's placed in the user GUI
  • GroupRankRequired:
    To make it work, you need to set up the options inside GroupRankRequired. If you enable it, only users with a rank on certain grup will be able to use it.
